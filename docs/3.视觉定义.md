以下是具体的视觉定义，你可以直接写入你的 tailwind.config.js。
🎨 1. 核心色板 (Color Palette)
车载 UI 最忌讳“刺眼”和“看不清”。我们采用 “深邃背景 + 霓虹点缀” 的方案。
 * 背景色 (Background): 深空灰 (Rich Dark)
   * 不建议用纯黑 (#000000)：除非你的屏幕是顶级 OLED，否则纯黑在夜晚会显现出屏幕背光的灰色，很难看。
   * 建议： 使用极深的蓝灰色。
 * 主品牌色 (Brand Primary): 星云紫 (Nebula Violet)
   * 寓意： 神秘、AI、高级感。紫色在夜晚驾驶时对视觉的刺激度低于红色（警告）和黄色（注意），相对安全。
 * 交互色 (Accent): 赛博青 (Cyber Cyan)
   * 用于高亮“原唱开启”或进度条，与紫色形成冷色调的赛博朋克对比。
💻 开发者配置 (直接抄作业):
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        // 1. 背景色：使用 Slate-950 或自定义的深空色
        background: '#020617', // 极深的蓝黑
        surface: '#0f172a',    // 卡片背景 (比背景稍亮)
        
        // 2. 品牌主色 (Violet)
        primary: {
          DEFAULT: '#8b5cf6', // Violet 500
          glow: '#7c3aed',    // Violet 600 (用于发光效果)
        },
        
        // 3. 辅助色 (Cyan)
        accent: '#06b6d4',    // Cyan 500

        // 4. 文本色 (高对比度)
        'text-main': '#f8fafc',  // 几乎纯白 (Slate 50)
        'text-muted': '#94a3b8', // 灰色 (Slate 400)
      }
    }
  }
}

🔠 2. 字体与排印 (Typography) - 车载特供版
车载屏幕距离人眼通常在 50cm 以上，且车辆会颠簸。网页的标准字号（14px/16px）在车上是绝对看不清的。
 * 字体家族： Inter, Roboto, 或系统默认 sans-serif。不要用衬线体。
 * 字号策略 (Type Scale)： 整体放大 1.25 倍。
| 元素 | Tailwind 类名 | 实际大小 | 场景 |
|---|---|---|---|
| 当前歌词 | text-5xl font-bold | ~48px | 沉浸模式的核心焦点 |
| 下一句歌词 | text-2xl text-muted | ~24px | 沉浸模式的辅助信息 |
| 页面标题 | text-3xl font-bold | ~30px | 顶部栏 "正在播放" |
| 列表歌名 | text-xl font-medium | ~20px | 首页列表 |
| 辅助标签 | text-base | ~16px | 歌手名、BPM |
💠 3. 形状与质感 (Shape & Texture)
 * 圆角 (Border Radius): 大圆角 (Large Rounded)
   * 定义： 使用 rounded-xl (12px) 或 rounded-2xl (16px)。
   * 理由： 汽车内饰（方向盘、仪表盘）大多是曲线。尖锐的直角在车内显得格格不入且具有攻击性。
 * 按钮形态: 胶囊型 (Pill Shape)
   * 使用 rounded-full。这样的按钮点击面积最大，最容易盲按。
 * 质感: 玻璃拟态 (Glassmorphism)
   * 极氪 8155 芯片 GPU 性能不错，我们可以在底部控制栏使用“毛玻璃”效果，透出底部的 MV 画面，非常高级。
   * CSS: bg-black/60 backdrop-blur-md (黑色半透明 + 背景模糊)。
🛠️ 4. 推荐工具库 (你的武器库)
不要从零写 CSS。请安装以下两个库，它们能让你在 1 天内搭出大厂级别的 UI：
 * Tailwind CSS: (必选) 原子类 CSS 框架。
 * shadcn/ui: (强烈推荐)
   * 这不是一个 npm 包，而是直接复制组件代码到你项目里。
   * 怎么用？
     * 搜索框： 复制它的 Input 组件，加高到 h-16。
     * 列表： 复制它的 Card 组件。
     * 进度条： 复制它的 Slider 组件，把轨道加粗。
 * Lucide React: 图标库。线条优美，完美适配 Tailwind。
👁️ 视觉预览 (Mental Render)
根据以上定义，你的 P0 播放控制栏 看起来会是这样：
> 一个悬浮在底部的黑色半透明长条（玻璃质感）。
> 中间有一条发着微弱紫色光晕的粗进度条。
> 左下角有一个胶囊形状的开关，关闭时是灰色，打开时亮起赛博青色的灯，显示 "VOCAL ON"。
> 正中间是一个巨大的白底黑标播放按钮（圆形），按下去会有微缩动画。
